# 经验规则库 (rules/)

把反复用到的、跨会话的经验沉淀成规则 —— 一个主题一个 md 文件。
新增后在 `../CLAUDE.md` 的「经验规则库」清单里登记（`@rules/<文件名>.md`），确保自动加载。

## 格式

```markdown
# <规则标题>

## 触发场景
什么时候该应用这条规则。

## 规则
具体怎么做（可分点）。

## 为什么
这条规则因何而生（通常是一次教训）。
```

## 原则

- 一条规则只管一个主题，标题即触发场景。
- 规则来自真实教训或反复确认的做法，不凭空造。
- 失效的规则及时删除或更新 —— 有规则却不遵守，比没有规则更糟。
